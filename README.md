DemoApi
======

## Synopsis

DempApi is a sample project to showcase how we can create REST APIs in Django-REST.

## Prerequisites

- Linux (Preferably Ubuntu 20.04)
- Python 3.6+ (Preferably 3.9)
- pip
- virtualenv
